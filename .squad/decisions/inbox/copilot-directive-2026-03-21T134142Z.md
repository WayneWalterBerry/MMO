### 2026-03-21T13:41:42Z: Objects crossing level boundaries

**By:** Wayne "Effe" Berry (via Copilot)
**What:** Objects may cross level boundaries — if a player picks up an object in one level they may take it to another level. If the level designer doesn't want an object to transfer to the next level (it might make a next level puzzle easier), then there MUST be a task/puzzle that destroys that object before they can enter the next level.
**Why:** User request — captured for team memory. Prevents future puzzles from being trivially solvable by objects carried from previous levels. Removal must be natural/diegetic.
