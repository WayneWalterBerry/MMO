### 2026-03-22T12:20: User directive — Container open/closed sensory rules
**By:** Wayne (Effe) Berry (via Copilot)
**What:**
- Containers can be open and closed (e.g., chest, drawer)
- **Closed container:** senses don't work to examine contents — can't look inside, can't feel inside
- **Open container:** all senses work on contents (search, feel, look, examine)
- **Exceptions:** some containers allow partial senses when closed (e.g., glass bottle — can see inside but not feel inside)
- **Chest:** new object to document — requires two hands to carry, can be open/closed
- **Nightstand drawer:** is a container, can be open/closed — update nightstand docs to reflect this
- This is a design rule: container state gates which senses can access contents
**Why:** User design notes — containers need proper open/closed sensory gating in both docs and engine
