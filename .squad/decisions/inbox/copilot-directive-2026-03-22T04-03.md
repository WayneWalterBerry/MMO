### 2026-03-22T04:03: User directive — Search/Find verb semantics
**By:** Wayne (Effe) Berry (via Copilot)
**What:** 
- `find` and `search` = use ALL senses (works in dark and light)
- `look` and `see` = vision only (requires light)
- `search around` in dark → "you feel out a nightstand" / in light → "you see a nightstand"
- `find the ticking` → uses hearing to locate the source
- `find` is the universal discovery verb — engine picks the appropriate sense based on context (dark=touch/hearing, light=vision+all)
- Once an object is discovered (e.g., nightstand found by touch), player can interact with its parts (e.g., "open drawer") even in darkness — they know it's there
- This is a core parser/engine distinction: search/find ≠ look/see
**Why:** User request — fundamental verb semantics for sensory interaction model
