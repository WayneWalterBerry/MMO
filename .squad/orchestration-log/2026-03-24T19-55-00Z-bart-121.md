## Key Work

1. **Proximity Model (3 Tiers)**
   - **SAME_SURFACE (0.8):** Items on the same furniture surface (high spread chance)
   - **SAME_PARENT (0.5):** Items on different surfaces of same furniture (medium spread)
   - **SAME_ROOM (0.2):** Items loose in room or different furniture (low spread)

2. **Spread Formula**
   - chance = proximity_factor × target_flammability × source_intensity
   - All values derived from material system
   - No per-object fire configuration needed

3. **Rate Limiting**
   - MAX_IGNITIONS_PER_TICK = 2
   - Fire cascades over multiple turns
   - Player always has at least one turn to react

4. **Generic Destruction Countdown**
   - Non-FSM objects: _burn_ticks_remaining = 1
   - Burn for one tick before destruction
   - Allows extinguishing before removal

5. **Burning vs Lit State**
   - Lit candles do NOT propagate fire
   - Only objects with state.is_burning = true spread fire
   - Prevents candles from being fire hazards

6. **Test Coverage**
   - Proximity factor calculations
   - Spread chance formulas
   - Rate limiting enforcement
   - Deterministic RNG with ctx.fire_rng
   - FSM vs generic object burning
   - Terminal state guards
