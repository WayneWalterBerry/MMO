       location = "matchbox",
       per_instance_overrides = {

2. **Key Design Decisions**
   - Self-contained module (own deep_copy/deep_merge)
   - instance_guid field (not guid)
   - Base.guid recorded as type_id
   - Pure Lua GUID generator (UUID v4 via math.random)
   - Sequential id naming ({prefix}-1, {prefix}-2, etc.)

3. **Core Principle 5 Support**
   - Multiple instances from single base template
   - Each instance gets unique GUID
   - Base object remains immutable
   - Per-instance overrides possible
   - Full traceability via type_id
