### Architecture Pattern: `register()`

```lua
-- in verbs/init.lua (thin dispatcher)
local verbs = {}
verbs.register = function(module_name, handlers_table)
    for verb, handler in pairs(handlers_table) do
        verbs[verb] = handler
    end
end

-- load spatial verbs
require("src.engine.verbs.spatial-verbs").register(verbs)
```

Each module exports `register(verbs_table)` function that populates the verb table.
