## Decision: D-SEARCH-ACCESSIBLE
When search opens a container via `containers.open()`, it must set `object.accessible = true` in addition to `is_open` and `open`. The `find_visible` function checks `obj.accessible ~= false` before searching container contents. Without this, discovered items remain invisible to verb handlers.
