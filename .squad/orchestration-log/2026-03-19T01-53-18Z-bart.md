## Key Architectural Decisions

1. **Object sources as mutation fuel** — All object `.lua` files loaded as raw strings at startup and indexed by object ID. Mutation variants exist as dormant strings until triggered.

2. **Verbs module factory pattern** — `verbs.create()` returns a handler table. Handlers receive `ctx` as parameter; clean dependency injection, no closures.

3. **Light via object properties** — `casts_light = true` (candle-lit) and `allows_daylight = true` (open curtains). Property checks in verb layer. Intentionally minimal.

4. **Real-time game clock** — `os.time()` delta × 24 = game seconds. Time always accurate between commands. Daytime 6 AM to 6 PM.

5. **Exit mutations use partial merge** — Exits modify fields in-place (distinct from object mutations which full-replace).

6. **Layered object search** — Finds objects in order: room → accessible surfaces → containers → inventory.

7. **Spawn deduplication** — Second instance of same spawn ID gets `-2`, `-3` suffix for unique IDs.
