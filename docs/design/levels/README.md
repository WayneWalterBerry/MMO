# Level Design Methodology

This folder contains the **design methodology** for how levels are created in the MMO — not individual level documents (those live in `docs/levels/`).

## What's Here

| Document | Description |
|----------|-------------|
| `level-design-considerations.md` | Design considerations and rules for level design |

## Relationship to Other Folders

| Folder | Contains |
|--------|----------|
| `docs/levels/NN/` | Level-specific documents (e.g., `01/level-01-intro.md`, `01/rooms/`, `01/puzzles/`) |
| `docs/design/levels/` | **This folder** — how to design levels (methodology, principles, considerations) |
| `docs/levels/NN/rooms/` | Room documentation specific to Level NN |
| `docs/levels/NN/puzzles/` | Puzzle documentation specific to Level NN |
| `docs/objects/` | Object specifications (**shared across all levels**, not level-specific) |
| `docs/design/rooms/` | Room design methodology |
