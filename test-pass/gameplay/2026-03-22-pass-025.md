## Analysis & Recommendations

### Root Causes
1. **Scoped search (BUG-079):** `search X` without a target doesn't traverse X's surfaces/containers. Only compound `search X for Y` triggers nested traversal.
2. **Article handling (BUG-081):** Parser strips "the"/"a" from scope words but not from target words in find/search operations.
3. **Embedding matcher hangs (BUG-086, BUG-087):** When Tier 1/2 parsing fails, the embedding matcher's fallback can enter infinite loops on certain inputs.
4. **Incomplete preprocessing (BUG-085):** Adverb list missing common words like "thoroughly", "quickly", "slowly".
5. **"look at" pattern (BUG-087):** Missing preprocess rule to convert "look at X" → "examine X".

### Priority Fix Order
1. **P0:** BUG-087 (`look at X` hang) — extremely common player phrase
2. **P0:** BUG-079 (scoped search empty) — core search feature broken
3. **P0:** BUG-080 (wardrobe search hang) — game-breaking
4. **P1:** BUG-081 (article stripping in targets) — affects all "find the X" patterns
5. **P1:** BUG-084 (abstract find hang) — affects compound commands
6. **P1:** BUG-086 (`check` hang) — common player verb
7. **P2:** BUG-083, BUG-085, BUG-082, BUG-078, BUG-088
