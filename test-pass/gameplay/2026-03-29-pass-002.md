## Blocked Tests (Require #429 + #456 fix)

The following test scenarios could NOT be executed due to empty instances and darkness:

1. **Happy path full sequence:** read riddle → touch answer → board lights green → repeat × 3 → trophy appears
2. **Wrong answer responses:** touch non-answer object → "That's not the answer to any riddle"
3. **Out-of-order solving:** solve riddle 3 before riddle 1 — does it allow or require sequence?
4. **FSM state tracking:** boards transition unsolved → solved, room progresses through states
5. **Trophy appearance:** after all 3 solved, trophy appears and is takeable
6. **Victory message:** "BRILLIANT THINKING, WYATT!" bell ring sequence
7. **Per-object sensory details:** all on_feel/on_smell/on_listen/on_taste for each of 8 objects
8. **Re-reading solved riddles:** board shows "already solved" or green state
9. **Keyword coverage:** "clock", "time", "watch" all resolve to arena-clock; "piano", "keys", "keyboard" resolve to arena-piano; "hole", "pit", "gap" resolve to stage-hole

## Recommendations

1. **Fix #429 FIRST** — Wire all 68 object files into their respective rooms via `instances` arrays with proper `type_id` GUIDs and nesting relationships.
2. **Fix #428 NEXT** — Add `light_level = 2` to all 7 Wyatt World rooms (or add `casts_light = true` to spotlight and wire it in).
3. **Re-run this playtest** after fixes to validate the actual puzzle mechanics.
4. **Consider:** Wyatt's World may need a different time-of-day start (e.g., 10 AM) or `always_lit` rooms since the darkness mechanic is inappropriate for an E-rated kid's game.
