### Minor Observations (NOT bugs):
- **Spent matches accumulate on floor** — "There is a spent match here. There is a spent match here. There is a spent match here." This is correct behavior but could be cleaned up with a count display later (e.g., "Three spent matches lie on the floor").
- **"light match" with spent match in hand** gives "You can't light a spent match" but doesn't auto-suggest dropping it. The GOAP "light candle" handles this correctly, but direct "light match" doesn't trigger GOAP to swap the spent match. Players would need to know to drop then get. Not a bug — GOAP is designed for goal-level commands like "light candle", not tool-level like "light match".

## Puzzle Feedback

### For Bob — Puzzle Experience Rating: ★★★★½ (4.5/5)

**Positives:**
1. **GOAP is MAGIC for candle relighting.** After burning through a match, "light candle" auto-plans the entire chain: drop spent → get fresh → strike → light candle. This is the single most satisfying moment in the game. It feels like the game understands you.
2. **Candle burn timer creates genuine urgency.** The candle dying right as I pulled the rug was *perfect* dramatic timing. It forces real decision-making about what to do while the light lasts.
3. **The spatial puzzle sequence is excellent.** push bed → pull rug → discover key AND trap door is a clean, satisfying chain of revelations.
4. **Sensory verbs are incredibly atmospheric.** The smell descriptions alone ("You'd rather not" for the chamber pot) add enormous personality.
5. **Pour vs drink distinction works perfectly.** A player who pours the poison is making a deliberate choice distinct from drinking it. This is good puzzle design.

**Suggestions:**
1. **Spent match clutter** — after heavy match use, the room description gets cluttered with "There is a spent match here" repeated 3+ times. Consider a grouped display ("Three spent matches lie scattered on the floor").
2. **"light match" with spent match** — A player who types "light match" while holding a spent one might expect the game to auto-swap. Currently only "light candle" triggers GOAP chain. Consider making "light match" also GOAP-aware (detect spent match in hand → drop → get fresh → light).
3. **Darkness after candle** — The cellar has no persistent light source yet. The torch bracket is present but not usable. This will be critical for Room 3 content.

**Bottom Line:** The bedroom escape is a polished, atmospheric experience. All 5 bug fixes land cleanly. The GOAP spent-match handling transforms what was a frustrating dead-end into the game's best moment. Ready for Room 3 content.
