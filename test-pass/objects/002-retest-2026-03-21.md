## 3. POISON BOTTLE — Pour vs Drink Disambiguation (BUG-102 Focus)

### Material: `glass` ✅ (registry verified)

### FSM Transitions

| # | Transition | Verb | Result | Status |
|---|-----------|------|--------|--------|
| 1 | sealed → open | `uncork bottle` | "You twist and pull the cork free with a soft pop. A wisp of sickly green vapor curls from the bottle's mouth." | ✅ PASS |
| 2 | open → empty | `pour bottle` | "You tip the bottle. The green liquid pours out, hissing where it touches the stone floor. A thin vapor rises, and then it is gone." | ✅ PASS |

### verb_hint Verification (Code Review)

The `fsm.transition()` function now accepts a 5th parameter `verb_hint`:
- The `pour` handler passes `verb_hint = "pour"` → matches the `pour` transition
- The `drink` handler passes `verb_hint = "drink"` → matches the `drink` transition
- Fallback logic: if verb_hint doesn't match, falls back to first available transition
- Alias checking included: verb_hint also matches against transition `aliases`

**Previously:** `pour bottle` triggered the DRINK message because the drink transition came first in the transitions array and there was no verb discrimination.

**Now:** Each verb handler tells the FSM which specific transition to use. Disambiguation works correctly.
